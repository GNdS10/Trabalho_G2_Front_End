## Sobre a API

As APIs disponibilizadas pelo professor estavam fora do ar durante o desenvolvimento. O professor autorizou o uso do **Supabase** como alternativa.

## Tecnologias

- React + Vite
- Google Maps API
- Supabase
- Netlify
